# Sales Performance Analysis (SQL)

## Project Overview
This project analyzes sales data using SQL to identify sales trends, customer purchasing behaviour and salesperson performance.

## Dataset
The dataset contains sales transactions with information such as:
- Customer Name
- Salesperson
- Product
- Category
- Region
- Oreder Date
- Sales Amount

## SQL Skills Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- TOP
- HAVING

## Analysis Performed
- View all sales data
- Calculate total sales
- Count total orders
- Calculate average sales
- Identify top 10 customers by purchases
- Analyze total sales by salesperson
- Analyze total sales by category
- Analyze total sales by region
- Identify best-selling products
- Rank products by sales
